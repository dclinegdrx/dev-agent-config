# OpenCode

OpenCode is an AI coding agent for the terminal. The recommended install path is Homebrew using the official OpenCode tap:

```sh
brew install anomalyco/tap/opencode
```

The OpenCode tap is preferred over the general Homebrew formula because it is usually updated more frequently.

## Configuration

The global OpenCode config usually lives at:

```text
~/.config/opencode/opencode.json
```

This repo keeps its shared config example in `opencode/opencode.json`.
